# React Routing Assignment

This is a React web application that demonstrates routing and functionality for a simple e-commerce store.

## Features

- Home page that loads by default when the app starts
- Navbar with sections: Home, Product, and Add to Cart
- Product page displays data fetched from an API in card format, including price, category, image, and an "Add to Cart" button
- Clicking the "Add to Cart" button stores the selected product data in the cart
- The Add to Cart section shows all the items added to the cart, including a "Remove Item" button for each item
